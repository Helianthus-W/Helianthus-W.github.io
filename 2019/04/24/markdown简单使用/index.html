无序列表

在列表前加上一个*、+、-即可，文字与符号之间最好空一格

  • 首先，我们应该
  • 其次，我们应当
  • 白日依山尽
  • 黄河入海流
  • 欲穷千里目
  • 更上一层楼

有序列表

添加有序列表的方法是，在文字前添加数字加英文句点，文字与符号之间最好空一格

  2. 《活着》
